üéØ Overview

Git Editor is a powerful command-line tool designed for rewriting Git commit history. It allows you to modify commit timestamps, author information, and messages with precision and safety.

‚ö†Ô∏è Important: This tool rewrites Git history permanently. Always backup your repository before use.

üíæ Installation

From Release Binaries

# Download the latest release for your platform
wget https://github.com/rohansen856/git-editor/releases/latest/download/git-editor-linux-x64.tar.gz
tar -xzf git-editor-linux-x64.tar.gz
sudo mv git-editor /usr/local/bin/

From Source

# Clone the repository
git clone https://github.com/rohansen856/git-editor.git
cd git-editor

# Build and install
cargo build --release
sudo cp target/release/git-editor /usr/local/bin/

üöÄ Operation Modes

Git Editor supports multiple operation modes to suit different use cases:

Mode Flag Description Use Case
Full Rewrite Default Rewrites entire repository history Complete timestamp and author updates
Show History -s, --show-history Display commit history without changes Repository analysis and planning
Pick Specific -p, --pick-specific-commits Interactive selection of individual commits Targeted commit modifications
Range Edit -x, --range Select and edit commit ranges Batch editing of consecutive commits
Simulation --simulate Preview changes without applying Safe testing and validation
Documentation --docs Open comprehensive documentation Help and reference

üí° Usage Examples

Open Documentation

git-editor --docs

Opens this comprehensive documentation in your default browser.

Basic Full Rewrite

git-editor --name "John Doe" --email john@example.com \
           --begin "2023-01-01 09:00:00" --end "2023-12-31 17:00:00"

Rewrites all commits with new author info and timestamps distributed between the specified dates.

Show Repository History

git-editor -s

Displays the current commit history without making any changes.

Simulation Mode

git-editor --simulate --show-diff \
           --name "Test User" --email test@example.com \
           --begin "2023-01-01 09:00:00" --end "2023-01-31 17:00:00"

Previews all changes with detailed diffs without applying them.

üîß Advanced Features

Smart Date Distribution

Git Editor uses intelligent algorithms to distribute commit timestamps:

  • Chronological Order: Maintains original commit sequence
  • Realistic Spacing: Distributes commits with natural intervals
  • Working Hours: Concentrates commits during business hours
  • Weekend Handling: Reduces weekend commit frequency
